# ⚡ Hogwarts Wizarding Portal

Welcome to the **Hogwarts Wizarding Portal**! This is a responsive web project built with **Bootstrap 5** and **Sass**, designed to bring the magic of the wizarding world to your browser. From the latest news in the *Daily Prophet* to the prestigious *Hogwarts Houses*, this portal is your gateway to everything magic.

## 🧙‍♂️ Key Features

* **Hogwarts Houses Section**: A dynamic display of the four legendary houses (Gryffindor, Slytherin, Ravenclaw, and Hufflepuff) featuring responsive cards and house-specific color schemes.
* **The Daily Prophet**: A clean, "newspaper-style" layout for wizarding news. It uses **Equal Height Cards** (`h-100`) to ensure a perfectly aligned grid, even when news snippets vary in length.
* **Diagon Alley Shop**: A dedicated page for wizarding commerce, showcasing iconic shops like Ollivanders and Weasleys' Wizard Wheezes with high-quality imagery and consistent ratios.
* **Interactive Carousel**: A sleek, auto-playing banner featuring iconic Hogwarts moments. Navigational controls are hidden by default and appear gracefully on hover for a cleaner UI.
* **Revelio Mode**: A secret feature (under development) to reveal hidden magical details within the site.

## 🛠️ Tech Stack

* **HTML5 & CSS3**: Core structure and advanced styling.
* **Bootstrap 5**: Utilized for the responsive grid system (`row-cols`), utility classes (spacing, flexbox), and UI components (Carousel, Navbar, Cards).
* **Sass (SCSS)**: For modular styling, variables (house colors), and cleaner nested CSS rules.
* **Google Fonts**: Integrated "Crimson Text" to capture the classic, old-world magical aesthetic.
* **pnpm**: Used as the fast, disk-efficient package manager for handling project dependencies.
